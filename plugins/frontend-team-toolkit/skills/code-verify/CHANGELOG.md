## v2.0.0 (2026-05-20)

### 哲学升级：先锚定，再迭代

技能从「七维验证工具」升级为 **AI 辅助第三方开发的全链路防偏方法论**。

**新增**:
- 哲学内核：三陷阱、三「不」、配对编程角色
- **五阶段闸门**：方案锚定 → 双锚验证 → 小步验证 → 止损换策 → 经验沉淀
- **参考实现锚**（阶段 2 扩展，超越仅对照文档）
- **小步验证 MVU 协议**（按文档章节拆分、失败协议）
- **止损点**：时间盒（30/45/60 个人经验值）+ 置信度闸门（≤4/7 禁止盲迭代）
- **检查清单模板** `templates/checklist-template.md`
- 全链路报告模板 `templates/report-template.md` 重构
- `validate.sh` 支持 `--ref-url`、`--phase`，生成五阶段骨架
- 企业微信 JS-SDK 降级为**附录案例**，非技能边界

**保留**:
- 七维验证引擎（嵌入阶段 2）
- 置信度等级表（与止损联动）
